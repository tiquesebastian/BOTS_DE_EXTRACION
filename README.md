# BOTS DE EXTRACCION

Scripts para extraer:
- Fechas de ingreso
- Numeros de documento

## Estructura
- FECHAS/fechas.py: extractor de fechas
- documentos/documentos.py: extractor mixto (TIF/PDF/imagenes)
- documentos/documentos_pdf.py: extractor solo PDF
- EXTRATOR/extractor_documentos.js: extractor alterno en JS

## Requisitos
Instalar dependencias Python en un entorno virtual.

## Nota
Este repositorio guarda codigo y utilidades. Los archivos de salida (csv/xlsx/log/zip) se excluyen con .gitignore.
